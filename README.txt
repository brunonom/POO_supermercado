Descrição:

Um projeto de programacao orientada a objetos que simula atraves do terminal um sistema de supermercado no qual:

		o supermercado possui:
				um nome
				uma colecao de setores
				uma colecao de pessoas
				uma colecao de caixas

		um setor possui:
				um nome
				uma colecao de produtos

		uma pessoa possui
				um nome
				um rg
				um cpf
				uma idade

				e pode ser do tipo
						cliente, que possui
								uma colecao de itens, dita compras
								um email

						funcionario, que possui
								um salario

								e pode ser do tipo
										atendente de caixa, que possui
												um caixa

										gerente, que possui
												um bonus fixo de 10% no salario

		um caixa possui
				um numero unico
				o cliente que esta sendo atendido nele

	onde cada atributo pode ser criado, removido, apresentado e editado, sempre respeitando as restricoes necessarias
